Unity

While on Unity in my class, we have to do a lot of assignments, and finally, I had to work with other people for a final project based on everything we learned on Unity. The theme of this project was surviving waves of enemies. On this game, I worked on creating scenes with buttons to go to the menu (did not create the sprite option or sprites in general), how to play description, and to the game itself. Also, I worked on the lose screen when the user dies from the enemies and created a timer for how long the user last in the waves on enemies. The user can reset or go back to menu after the game is over too by clicking on the buttons that appear when the user dies in the game. To add on to this, I added an enemy spawner to make sure that enemies keep on spawning after they are destroyed by the user. Finally, I added collision on when the user hits the enemies and when the enemies hit the user. The music (actually sounds) the user hears while playing the game is actually from me too!
